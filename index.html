🔔 Powiadomienia push

Otrzymuj powiadomienia o nowych wydatkach i przychodach – nawet gdy przeglądarka jest zamknięta lub ekran telefonu jest wyłączony.

⚠️ Twoja przeglądarka nie obsługuje powiadomień push.
⚙️ Wymagana konfiguracja: Uzupełnij klucz VAPID w src/modules/pushNotifications.js i wdróż Firebase Cloud Functions (cd functions && npm install && firebase deploy --only functions). Szczegóły w README.md.

📊 Eksport danych do analizy LLM

Eksportuj kompletne dane budżetowe w formacie przystosowanym do analizy przez modele AI (ChatGPT, Claude, itp.). Eksport zawiera wszystkie szczegóły: przychody, wydatki, kategorie, limity, historię transakcji z dokładnym osadzeniem w czasie.
